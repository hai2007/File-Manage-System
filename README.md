## 如何使用？

打开```eclipse```后，按照下面顺序导入下面：

```
[File] -> [Import] -> [Maven] -> [Existing Maven Projects]
```

然后选择当前项目即可成功导入。

导入后运行只需要把```Main.java```作为入口，选择```run as Application```即可。

此时，服务器就启动了，由于采用了前后端分离开发模式，你还需要启动Web端。

关于Web端，请打开```src/main/webapp/README.md```查看说明。
